### 🚀How to use
В меню Meteor Client должен появится новый столб и там все функции которые я добавил на сейчас их 3. позже будет больше
1 функция: TntGrief. это функция требует креатив. она будет ставить тнт и поджигать автоматически.
2 функция: LavaGrief. это как с  tnt-grief но только лава.
3 функция: ForceCreative. это функция включает всем креатив кроме хоста и тебя. (толку нет)
4 функция: CrashPlayers. эта функция в разработке.

всего функций будет где то 35-50.
